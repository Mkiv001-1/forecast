# Account Dashboard (Дашборд прибыльности)

## Цель
Визуализировать общую прибыльность и динамику портфеля по данным IB Gateway. Предоставить детализацию по инструментам и фильтры по периоду.

## Основные элементы
- **Агрегированные показатели (верх):**
  - Equity (стоимость портфеля)
  - Net Liquidation Value
  - Unrealized PnL
  - Realized PnL
  - Free Margin
  - Used Margin
  - Доходность за период (%)
- **График equity curve**
- **Таблица истории (низ):**
  - Дата/время
  - Тикер
  - Equity
  - Unrealized PnL
  - Realized PnL
  - Cumulative PnL
  - Объем
  - Цена
  - Комментарии (опционально)

## Фильтры
- По периоду: 1 день, неделя, месяц, произвольный интервал
- По инструменту (тикеру)

## Источник данных
- Только через IB Gateway API (core/ib_gateway_client.py)
- История хранится в SQLite (таблица portfolio_history)

## Взаимодействие
- Регулярный сбор snapshot-данных (по расписанию/триггеру)
- Агрегация и расчет cumulative PnL
- Экспорт таблицы (CSV/XLSX)

## Визуализация
- PyQt6: QTableView (таблица), QChart (график), фильтры (QComboBox/QDateEdit)
- Цветовая индикация прибыли/убытка

## Верификация
- Сравнение с IB TWS/Account Management
- Проверка фильтров и расчетов
- Юнит-тесты на агрегацию и фильтрацию
